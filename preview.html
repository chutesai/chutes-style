Typography

Most of the personality comes from the contrast between a strong display face, a neutral body face, and a tidy mono face.

Display / Headlines

Tomato Grotesk carries the brand.

Use for headlines, nav, buttons, badges, and product labels. Keep tracking tight and let size do the work.

Body / Long-form

Neue Haas Unica stays readable and understated, which is why it works well underneath the louder display moments. Use it for paragraphs, descriptions, help copy, and docs.

Mono / Code / Data

const route = "/app/api";
const accent = "#8BFFC6";
const status = "LIVE";

Use the mono face for code blocks, key-value data, shell commands, and structured labels.

Logos and Assets

Brand marks are simple and clean. Partner logos sit inside softened bands. Provider logos work best in grayscale when they appear as a set.

Primary mark

Chutes

Secondary emblem for compact uses.

Favicon

Motion and Imagery

The brand leans on ambient motion, dark overlays, and a small set of recurring stills: infrastructure loops, product posters, and a few editorial photographs.

Homepage hero motion

Use for the biggest brand statement. Always pair with a dark gradient wash.

Platform motion

Best for infra, network, or decentralization stories.

Hyperscale photography

Editorial stills should feel graphic, moody, and slightly futuristic.

Product stills

Use rounded containers and keep the surrounding chrome dark and quiet.

Most reusable Chutes pages can be built from the same recipe: ink canvas, Tomato Grotesk headlines, Neue Haas body copy, moss as the active accent, one memorable motion asset, and restrained use of the gradient for breakthrough moments only.
